# ⛳ Golf Card Game

A beautiful, multi-player web-based implementation of the classic **Golf** card game (6-card variation). Play against three AI opponents (Chloe, Isaac, and Terri) across 9 holes of fun!

## 🎮 How to Play

The goal of Golf is to have the **lowest score** at the end of 9 holes.

### Scoring
| Card | Points |
| :--- | :--- |
| **Aces** | 1 point |
| **2s** | -2 points |
| **3-10** | Face value |
| **Jacks/Queens** | 10 points |
| **Kings** | 0 points |
| **Pairs** | 0 points (same rank in a vertical column) |

### Gameplay Summary
1.  **Start**: Each player has 6 cards face down. Flip over any 2 cards to begin.
2.  **Turn**: Draw a card from the deck or the discard pile.
3.  **Action**: Either swap the drawn card with one of your 6 cards or discard it (if drawn from the deck).
4.  **End of Hole**: When a player flips all their cards, everyone else gets one final turn.
5.  **Winning**: The player with the lowest total score after 9 holes wins!

## 🚀 Getting Started

Simply open the `index.html` file in any modern web browser to start playing.

```bash
# Path to the game
./index.html
```
